# HR Analytics - Employee Attrition Prediction

A machine learning-powered web app that predicts whether an employee is likely to leave or stay.

## Tech Stack
- **Frontend:** Streamlit
- **ML Model:** Random Forest Classifier (Scikit-Learn)
- **Data:** IBM HR Analytics Employee Attrition Dataset

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run employee_attrition.py
```

## Features
- Interactive sliders for input parameters
- Real-time attrition prediction with confidence score
- Clean dashboard layout with sidebar
